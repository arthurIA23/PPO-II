Joguinho, pq preciso passar de ano


------------------------- Tutorial para rodar o game ------------------------- 

1 - Instale em sua máquina a Unity Hub
2 - Após instalar a unity hub
3 - Instale a versão gráfica 6001.1.10f1
4 - Importe a pasta que vc vai instalar nesse repositório
5 - Abra o projeto e aproveite


------- Rodar usando o vscode ---------

Com o projeto aberto vá em Files -> Preferences -> External Tools -> Em "External script editor" coloque o visual studio code
Use as extensões: .NET install tool, c# da microsoft e c# dev kit, unity "integrates visual studio code"
